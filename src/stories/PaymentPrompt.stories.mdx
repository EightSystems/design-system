# PaymentPrompt

Um componente que agrupa os átomos `<Avatar>` e `<CurrencyLabel>` com uma string de texto em um único container. Pode
ser utilizado para mostrar a mensagem do usuário durante o link de pagamento, eu confirmar a ação ao final do fluxo de
pagamento.
