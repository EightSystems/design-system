# Tematização

## Webfonts

Por padrão, o design-system utiliza as fontes **Poppins** e **Inter**. As fontes estão disponíveis a partir do
`ThemeProvider` injetado na raiz do projeto, e pode ser usado em qualquer componente do projeto através das variáveis
CSS `--fontFace-Inter` e `--fontFace-Poppins`.

```css
p {
    font-family: var(--fontFace-Inter);
}
```
