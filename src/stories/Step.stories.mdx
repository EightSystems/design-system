# Step

> **Importante**: Esse componente faz parte de um componente composto e não foi projetado para ser usado de maneira independente.

Um componente que representa uma etapa em uma conjunto de etapas do componente `<Stepper>`. Deve receber obrigatoriamente as propriedades `stepLabel`, `stepIsFirst`, e `stepIsActive`.

<Canvas>
    <Story name="Step (Não-ativo)" args={{ stepLabel: "Confirmação de pagamento", stepIsFirst: true, stepIsActive: false }}>
        {args => <Step {...args} />}
    </Story>
</Canvas>

<Canvas>
    <Story name="Step (Ativo)" args={{ stepLabel: "Confirmação de pagamento", stepIsFirst: true, stepIsActive: true }}>
        {args => <Step {...args} />}
    </Story>
</Canvas>

Quando a propriedade `stepIsFirst` é `false`, o componente renderizará um conector no lado esquerdo:

        {args => <Step {...args} />}
    </Story>
