## Avatares com imagem

A propriedade padrão `src` pode ser usada para definir a fonte de uma imagem.

<Canvas>
    <Story name="Avatar with Image" args={{size: 'medium', src: 'https://images.unsplash.com/photo-1438761681033-6461ffad8d80?ixlib=rb-0.3.5&q=80&fm=jpg&crop=faces&fit=crop&h=200&w=200&s=046c29138c1335ef8edee7daf521ba50'}}>
        {args => <Avatar {...args} />}
    </Story>
</Canvas>

## Avatares com letras

Nas situações em que a imagem não está disponível (quando o usuário ainda não definiu uma imagem de perfil ou houve um
erro ao carregar a imagem), as letras iniciais podem ser passadas como uma `string` para `children`.

<Canvas>
    <Story name="Avatar without Image" args={{size: 'medium', children: ['EL']}}>
        {args => <Avatar {...args} />}
    </Story>
</Canvas>
