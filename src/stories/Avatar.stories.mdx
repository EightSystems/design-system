## Avatares com imagem

A propriedade padrão `src` pode ser usada para definir a fonte de uma imagem.

## Avatares com letras

Nas situações em que a imagem não está disponível (quando o usuário ainda não definiu uma imagem de perfil ou houve um erro ao carregar a imagem), as letras iniciais podem ser passadas como uma `string` para `children`.

<Canvas>
    <Story name="Avatar without Image">
        <Avatar size="medium">EL</Avatar>
    </Story>
</Canvas>
