# CurrencyLabel

Um wrapper para a representação de valores monetários, converte um valor puro na unidade monetária requerida de acordo
com a API de internacionalização.

<div className="flex-container">
    <CurrencyLabel currency="BRL" value={123.456} />
</div>

## Variantes

O `CurrencyLabel` usa a API de internacionalização `Intl.NumberFormat`, e possui duas propriedades obrigatórias:
`currency` e `value`:

### Real Brasileiro

<Canvas>
    <Story name="Real Brasileiro" args={{ currency: "BRL", value: 123.456 }}>
        {args => <CurrencyLabel {...args} />}
    </Story>
</Canvas>

### Dólar americano

<Canvas>
    <Story name="Dólar Americano" args={{ currency: "USD", value: 456.789 }}>
        {args => <CurrencyLabel {...args} />}
    </Story>
</Canvas>

### Yen Japonês

<Canvas>
    <Story name="Yen Japonês" args={{ currency: "JPY", value: 3456.789 }}>
        {args => <CurrencyLabel {...args} />}
    </Story>
</Canvas>

Adicionalmente, é possível especificar o tamanho a a unidade de medida para alterar o tamanho de exibição de
`CurrencyLabel` com `size` e `sizeUnit`.

### Euro com fonte em tamanho 72px

</Canvas>
