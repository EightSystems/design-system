# Button

The `Button` component is used to trigger an action or event, such as submitting a form, opening a dialog, canceling an
action, or performing a delete operation.

## Import

```jsx
import { Button } from "@8sistemas/design-system";
```

## Usage

<Canvas>
    <Story name="Primary Button" args={{ color: "primary", children: ["Primary Button"] }}>
        {args => <Button {...args} />}
    </Story>
</Canvas>

## Button Variants

Use the `hasOutline` prop to determinate if the button will be of the outlined variant.

<Canvas>
    <Story name="Primary Button (Outline)" args={{ color: "primary", hasOutline: true, children: ["Outlined Button"] }}>
        {args => <Button {...args} />}
    </Story>
</Canvas>

Use the `color` prop to determinate the color of the button using the `Theme` available primary colors.

<Canvas>
    <Story name="Danger Button" args={{ color: "danger", children: ["Danger Button"] }}>
        {args => <Button {...args} />}
    </Story>
</Canvas>
