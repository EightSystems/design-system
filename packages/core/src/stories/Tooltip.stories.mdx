# Tooltip

O tooltip é uma mensagem breve e informativa que aparece quado um usuário interage com um elemento. Tooltips geralmente
são iniciados ao interagir com um elemento.

<Canvas>
    <Story name="Tooltip" args={{ tooltipContent: "A vida é bela", placement: "bottom", offset: [0, 8] }}>
        {args => (
            <Tooltip {...args}>
                <Button>Click me for a surprise</Button>
            </Tooltip>
        )}
    </Story>
</Canvas>

## Import

```javascript
import { Tooltip } from "@8sistemas/design-system";
```

## Uso

O Tooltip deve envolver o componente com a qual será necessário interagir, e fornecer uma string ou elemento válido
React na propriedade `tooltipContent`. Também é requerido usar as propriedades `placement` e `offset` para determinar o
posicionamento do Tooltip em relação ao elemento ao qual ele está ancorado.

```jsx
<Tooltip placement="bottom" offset={[0, 8]} tooltipContent={<TooltipContent />}>
    <Button color="primary">Test</Button>
</Tooltip>
```
