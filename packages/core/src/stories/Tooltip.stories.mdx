# Tooltip

Tooltip shows as a quick and info message when the user interacts with the element. This can either be a click or a
hover event, but be aware that on native the only event available is the click event.

## Import

```javascript
import { Tooltip } from "@8sistemas/design-system";
```

## Usage

Tooltip must wrap the component which will trigger the event, and set a string, or a valid react element as the
`tooltipContent` prop. You must also set the `placement` and `offset` props so it know where to anchor the tooltip based
on the element.

```jsx
<Tooltip placement="bottom" tooltipContent={<TooltipContent />}>
    <Button color="primary">Test</Button>
</Tooltip>
```
