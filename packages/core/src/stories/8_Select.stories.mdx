# Select

A wrapper component and form control that allows the selection of options. It's the fusion of the elements `label` and
`select`.

This component is commonly used in forms and dialog boxes.

## Import

```jsx
import { Select, Option } from "@8sistemas/design-system";
```

## Select Variants

The `Select` component has three obligatory props, `name`, `label`, `children`.

<Canvas>
    <Story name="Regular Select" args={{ name: "selectName", label: "Select Label", placeholder: "Placeholder here" }}>
        {args => (
            <Select {...args}>
                <Option value="1">Default Option</Option>
                <Option value="2">Option 2</Option>
                <Option value="3">Option with a very large title to make it break</Option>
            </Select>
        )}
    </Story>
</Canvas>

Validation states from the components can be achieved by using the prop `validationError` to indicate a validation
error, and `validationSuccess` to indicate validation success, and `validationError` to show a error message bellow the
input.
