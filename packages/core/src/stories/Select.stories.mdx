# Select

A wrapper component and form control that allows the selection of options. It's the fusion of the elements `label` and
`select`.

This component is commonly used in forms and dialog boxes.

## Import

```jsx
import { Select } from "@8sistemas/design-system"
```
## Select Variants

The `Select` component has three obligatory props, `name`, `label`, `children`.

<Canvas>
    <Story name="Regular Select" args={{ name: "selectName", label: "Select Label" }}>
        {args => (
            <Select {...args}>
                <option value="1">Default Option</option>
                <option value="2">Option 2</option>
            </Select>
        )}
    </Story>
</Canvas>

Validation states from the components can be achieved by using the prop `validationError` to indicate a validation error, and `validationSuccess` to indicate validation success, and `validationError` to show a error message bellow the input.

### Validação com erro

<Canvas>
