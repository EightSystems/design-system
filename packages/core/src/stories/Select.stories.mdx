# Select

A wrapper component and form control that allows the selection of options. It's the fusion of the elements `label` and
`select`.

This component is commonly used in forms and dialog boxes.

## Variants

O `Select` requer três propriedades obrigatórias, `name`, `label`, e `children`:

<Canvas>
    <Story name="Regular Select" args={{ name: "selectName", label: "Select Label" }}>
        {args => (
            <Select {...args}>
                <option value="1">Default Option</option>
                <option value="2">Option 2</option>
            </Select>
        )}
    </Story>
</Canvas>

Variações de validação do componente podem ser alcançadas usando propriedades como `validationError` para indicar um
erro de validação, e `validationSuccess` para indicar uma validação de sucesso, e `validationMessage` para inserir uma
mensagem de ajuda durante a validação:

### Validação com erro

<Canvas>
    <Story
        name="Error Select"
        args={{
            name: "selectName",
            label: "Select Label",
            validationError: true,
            validationMessage: "There seems to be something wrong with your select",
        }}
    >
        {args => (
            <Select {...args}>
                <option value="1">Default Option</option>

            </Select>
        )}
    </Story>
</Canvas>
