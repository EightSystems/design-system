# Image

A replacement to `img` on Web and `Image` on Native. You should be using this component instead of img and Images, as it
will allow your code to be platform agnostic. That way you can simply do:

```jsx
const MyImageComponent = styled(Image)`
    border-radius: 50%;
`;
```

And it will work on **both** the platforms, and you can either send an URL, or a simple `require('./image.png')`

### Native Info

On native we use `FlexImage` taken from this repository here: https://github.com/kodefox/react-native-flex-image so it supports images without a fixed width and height

## Import

```jsx
import { Image } from "@8sistemas/design-system";
```
