# List

A simple List element with a ListItem composite, this will use FlatList on Native and a simple div with overflow-x auto
on web.

## Import

```jsx
import { List, ListItem } from "@8sistemas/design-system";
```

## Examples

### Quick List

<Canvas>
    <Story name="List">
        {args => {
            return (
                <List
                    {...args}
                    data={[1, 2, 3, 4]}
                    keyExtractor={item => `key-${item}`}
                    renderItem={({ item }) => {
                        return (
                            <ListItem
                                title={`Title ${item}`}
                                subTitle={`This is my sub-title ${item}, and it might overflow to the next line`}
                                text={`Really big text ext here ${item}, for helping understand a big list item`}
                                icon={"fa-heart"}
                                rightIcon={"fa-arrow-right"}
                            />
                        );
                    }}
                />
            );
        }}
    </Story>
</Canvas>
