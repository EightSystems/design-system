# Box

A replacement to `div` on Web and `View` on Native.
You should be using this component instead of divs and Views, as it will allow your code to be platform agnostic.
That way you can simply do:

```jsx
const MyContainerComponent = styled(Box)`
    padding-left: 100px;
`
```

And it will work on **both** the platforms

## Import

```jsx
import { Box } from "@8sistemas/design-system";
```
