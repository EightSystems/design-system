# Avatar

The `Avatar` component is used to represent a user, and displays the profile picture, initials or fallback icon.

## Import

```jsx
import { Avatar } from "@8sistemas/design-system";
```

## Usage

### Avatars with image

The default prop `src` can be used to point to a image source.

<Canvas>
    <Story
        name="Avatar with Image"
        args={{
            size: "medium",
            src: "https://images.unsplash.com/photo-1438761681033-6461ffad8d80?ixlib=rb-0.3.5&q=80&fm=jpg&crop=faces&fit=crop&h=200&w=200&s=046c29138c1335ef8edee7daf521ba50",
            alt: "Imagem de perfil: Nome do usuário",
        }}
    >
        {args => <Avatar {...args} />}
    </Story>
</Canvas>

### Avatar with letters

Nas situações em que a imagem não está disponível (quando o usuário ainda não definiu uma imagem de perfil ou houve um
erro ao carregar a imagem), as letras iniciais podem ser passadas como uma `string` para `children`.

In situations where the image is not available (user has yet to define the profile image or there was an error while
loading it from the server), the initial letters can be passes as a `string` to `children`.

<Canvas>
    <Story name="Avatar with letters" args={{ size: "medium", children: ["EL"] }}>
        {args => <Avatar {...args} />}
    </Story>
</Canvas>
