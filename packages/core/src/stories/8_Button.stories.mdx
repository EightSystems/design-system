# Button

The `Button` component is used to trigger an action or event, such as submitting a form, opening a dialog, canceling an
action, or performing a delete operation.

## Import

```jsx
import { Button } from "@8sistemas/design-system";
```

## Usage

<Canvas>
    <Story
        name="Primary Button"
        args={{ backgroundColor: "primary", children: ["Primary Button"], onClick: action("onClick") }}
    >
        {args => <Button {...args} />}

## Button Variants

You can use the various `borderColor`, `backgroundColor`, and `textColor` for creating variants on your self. For
example, see an outline button sample

<Canvas>
    <Story
        name="Primary Button (Outline)"
        args={{
            backgroundColor: "lightTint",
            textColor: "primary",
            borderColor: "primary",
            children: ["Outlined Button"],
            onClick: action("onClick"),
        }}
    >
        {args => <Button {...args} />}
    </Story>
</Canvas>

## Gradient Support

We support basic gradient background, you can use the `backgroundColor` prop with one of the `*Gradient` values. For
example, `primaryGradient`. Be aware that it's a simple linear-gradient.
