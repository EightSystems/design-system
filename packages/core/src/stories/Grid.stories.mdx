# Grid System

A Simple Grid System

We use a simple 12 columns grid system, with the default for each of the props being 12

### Breakpoints

Our breakpoints list are:
* xs: <= 376px
* sm: <= 768px
* md: <= 1000px
* lg: >= 1000px

***Small note: these pixel values are converted to EM at Web***
