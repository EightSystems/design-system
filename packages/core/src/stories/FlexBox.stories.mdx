# Flex Box / Centered Flex Box

This is a simple shortcut to a `Box` with:

```css
display: flex;
```

Same applies to the `CenteredFlexBox`, it's basically:

```css
display: flex;
align-items: center;
justify-content: center;
```

And it will work on **both** the platforms

## Import

```jsx
import { FlexBox, CenteredFlexBox } from "@8sistemas/design-system";
```
