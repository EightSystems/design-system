# ProgressRing

Component used to indicate progress. It's part of the component group that composes the [Stepper](), and can be
individually used as an ProgressIndicator, or a spinner.

<Canvas>
    <Story name="ProgressRing" args={{ radius: 30, stroke: 4, strokeColor: "gray", progress: 50 }}>
        {args => <ProgressRing {...args} />}
    </Story>
</Canvas>

Adicionally it's also possible to define props like `text`, `textColor`, `textXAxis`, `textYAxis` and `textFontSize` to
define and position a string of text inside the circle.

<Canvas>
    <Story
        name="ProgressRing with text"
        args={{
            radius: 34,
            stroke: 4,
            strokeColor: "gray",
            progress: 75,
            text: "3 de 4",
            textColor: "gray",
            textXAxis: 17,
            textYAxis: 38,
            textFontSize: 12,
        }}
    >
        {args => <ProgressRing {...args} />}
    </Story>
</Canvas>
