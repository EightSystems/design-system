# Text

The `Text` component is part of the typography module, you can use it as a substitute to `p` HTML tags. It will also replace React Native `Text` component.

You should use this component everytime you need a `Text` element, as it will make your component platform type agnostic.

## Import

```jsx
import { Text } from "@8sistemas/design-system";
```
