# CurrencyLabel

This is a monerary values wrapper, it would convert whole integers into the set currency using the Internationalization
APIs. That way you can set values without the worry of the decimal places and it would format it automatically for you.

Financial applications usually work with whole integers instead of Decimals for representing monetary values.

## Native Support

On native you would need to use the `intl jscFlavour`, changing this line in `android/app/build.gradle` in your project
from:

```
def jscFlavor = 'org.webkit:android-jsc:+'
```

to:

```
def jscFlavor = 'org.webkit:android-jsc-intl:+'
```

And, you also need to disable Hermes, Hermes doesn't support Intl (on Android, as for version 0.64.2)
